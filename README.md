# Nong-AI
Nong is a Raspberry Pi based AI companion inspired by the idea of coexistence between humans and AI.

Current Features:
- Offline Wikipedia knowledge base using Kiwix
- Local AI experimentation
- Speech recognition research
- Text-to-speech integration
- Modular hardware architecture

Planned Features:
- Camera vision
- Servo-based head movement
- Custom personality engine
- Fully offline operation

Hardware:
- Raspberry Pi 4B
- Microphone
- Camera
- Servo motors

Status:
Work in Progress
